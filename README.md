# HOW TO RUN

1. Config environment variables

`PORT`,
`NODE_CODEPUSH_DB_NAME`,
`NODE_CODEPUSH_DB_HOST`,
`NODE_CODEPUSH_DB_PORT`,
`NODE_CODEPUSH_DB_USER`,
`NODE_CODEPUSH_DB_DRIVER`,
`NODE_CODEPUSH_DB_PASSWORD`,
`NODE_ENV`

2. Install dependencies

```sh
npm install
```

or

```sh
yarn install
```

3. Run

```sh
npm run install
```

or

```sh
yarn run start
```
